Uma calculadora básica e funcional desenvolvida para web, aplicando conceitos fundamentais de manipulação do DOM e lógica de programação com JavaScript.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais da web:

- **HTML5:** Estruturação dos elementos da calculadora.
- **CSS3:** Estilização, layout e responsividade.
- **JavaScript:** Lógica das operações matemáticas e interação com o usuário.

---

## 📂 Estrutura do Projeto

```text
CalculadoraJavaScript/
├── Calculadora JavaScript/   # Pasta com os arquivos fonte
│   ├── index.html            # Estrutura principal
│   ├── style.css             # Estilização visual
│   └── script.js             # Lógica em JavaScript
├── Calculadora JavaScript.rar # Arquivo compactado do projeto
└── README.md                 # Documentação
```

---

🎯 Funcionalidades

➕ Operações básicas (Soma, Subtração, Multiplicação e Divisão).

🧹 Botão de limpeza (Clear).

📱 Interface responsiva para uso em diferentes dispositivos.

⚡ Resultados em tempo real.

---

⚙️ Como Executar

Não é necessário instalar nada! Para testar a calculadora:

Clone o repositório ou baixe os arquivos.

Navegue até a pasta Calculadora JavaScript.

Abra o arquivo index.html em qualquer navegador (Chrome, Edge, Firefox).
